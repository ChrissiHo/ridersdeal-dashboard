# 📋 RidersDeal DATEV-Abgleich Dashboard

Interaktives Lizenz- und Vertragsabgleich-Dashboard für die Buchhaltung.

## Setup

### 1. Supabase Tabellen anlegen
Im Supabase SQL Editor: Inhalt von `schema.sql` ausführen

### 2. Daten importieren
Im Supabase SQL Editor: Inhalt von `seed.sql` ausführen

### 3. GitHub Pages aktivieren
Settings → Pages → Source: GitHub Actions

### 4. Fertig
Die URL lautet: `https://[username].github.io/ridersdeal-dashboard`

## Features
- ✏️ Inline-Bearbeitung aller Felder
- 💬 Notizen & Kommentare (in Supabase gespeichert)
- 📎 PDF-Upload direkt beim Eintrag
- 👥 Mehrere Mitarbeiter gleichzeitig
- 🔄 Änderungen sofort sichtbar für alle
